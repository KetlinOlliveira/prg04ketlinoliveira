# 🐕 FindPet | Sistema de Adoções de Animais de Estimação

Plataforma interativa para adoção responsável de animais legalizados 

</div>

---

O **FindPet** é um sistema web voltado para a adoção consciente e responsável de animais de estimação legalizados. O projeto busca conectar pessoas a animais que precisam de um lar, oferecendo todas as informações necessárias para uma adoção segura — tanto para o tutor quanto para o animal.

> ⚠️ A venda de animais é ilegal. O FindPet existe para facilitar adoções, nunca comercializações.

---

## 🕰️ Funcionalidades Planejadas

- **Feed de Adoção** — Listagem de animais disponíveis com ficha completa (espécie, idade, temperamento, localização, etc.)
- **Quiz de Compatibilidade** — Algoritmo/IA para descobrir qual pet combina com você, levando em conta ambiente familiar, condição financeira e local de moradia
- **Chatbot de Dúvidas** — Assistente para responder perguntas sobre adoção, legalização e cuidados
- **Guia de Cuidados Especializados** — Informações detalhadas por espécie: habitat ideal, alimentação, higiene, saúde e muito mais
