## 🐕 Sobre o Projeto

O **FindPet** é um sistema web frontend desenvolvido com foco na **adoção responsável de animais de estimação**.

A ideia do projeto é criar uma plataforma acolhedora, acessível e informativa, capaz de aproximar pessoas interessadas em adotar de animais que precisam de um lar seguro.

O projeto começou como uma aplicação simples utilizando **HTML, CSS e JavaScript puro**, desenvolvida durante as atividades da disciplina de **Desenvolvimento Web**. Posteriormente, foi refatorado para **React**, aplicando conceitos de componentização, organização de pastas, navegação com React Router DOM e boas práticas de desenvolvimento frontend.

> ⚠️ **A venda de animais é ilegal.**  
> O FindPet existe para facilitar adoções responsáveis, nunca comercializações.

---

## 💛 Propósito

Mais do que um sistema, o FindPet representa uma proposta de conscientização.

A adoção de um animal envolve responsabilidade, cuidado, paciência e amor. Por isso, o projeto busca transmitir visualmente uma sensação de acolhimento, confiança e afeto, usando uma identidade visual baseada em tons terrosos, suaves e humanos.

O objetivo é que a experiência do usuário seja leve, bonita e informativa, mas também responsável.

## 🎨 Identidade Visual

A palheta de cores do FindPet foi pensada para transmitir:

- acolhimento;
- confiança;
- responsabilidade;
- carinho;
- proximidade com o universo pet.
