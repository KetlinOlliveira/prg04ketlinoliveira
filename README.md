  ## 🐈​​ FindPet | Sistema de Adoções de Animais de Estimação
 
 </div>
 
 Plataforma interativa para adoção responsável de animais legalizados. O projeto irá contar com um chatbot para recomendação de pets ideais e um guia completo de cuidados específicos para diversas espécies.


---


  -  O projeto utiliza uma organização modular: a pasta infrastructure/assets centraliza recursos como estilos (CSS), scripts (JS), ícones e imagens, enquanto a pasta pages gerencia as visualizações da aplicação, garantindo um código limpo e de fácil manutenção.
